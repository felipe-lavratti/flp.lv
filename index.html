Welcome to my on-line spot.
I'm Felipe, an Embedded Software Engineer, got my Master degree in microelectronics, and graduated in electrical engineering, yet, my thing is software. You are invited to mess around this site. Check my online profiles, open source projects, blogs and video channels.

Currently I'm putting a lot of effort on helping developers to learn better firmware principles.

Through an Youtube channel I talk about embedded software development and teach automated testing, the content is presented in Portuguese.

I periodically write articles to Embarcados portal, also in Portuguese.

There's also a few extra repositories in Github showing unit testing in embedded C with CppUTest, multi-platform build systems using Scons and Makefiles, and more to come.
